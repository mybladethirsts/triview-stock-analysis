# Triview — 三视股票分析（TriView Stock Analysis）

三维独立评分 · 行业自适应 · 加权共识 · 机械自检

## 核心理念

> **一只股票，三个完全独立的视角来照。不靠单一方法论，不靠单一行业的经验。**

架构由一个核心问题和三条原则驱动：

**任何股票，无论什么行业，都可以从三个层面来分析：**
1. 这家公司的生意本身好不好？（质地）
2. 现在的大环境是否有利于它？（宏观）
3. 市场信号告诉你什么？（技术）

第一维会自动根据行业分类路由到对应的评分分支，无需事先指定行业。

## 适用场景

当用户提到以下内容时激活此技能：
- 股票分析、投资研究、市场分析
- 美股/港股/A 股具体标的（如 NVDA、JPM、0700.HK、600519.SS）
- 公司护城河/竞争优势分析
- 供应链卡点/卡脖子/瓶颈环节分析（制造业科技股专用）
- 宏观周期判断与行业轮动
- 技术入场时机、止损设置
- "这只股票怎么样"、"现在该不该买/卖"
- **任何行业**：消费、金融、医疗、平台、能源、制造

## 核心框架：三维独立评分

| 视角 | 权重 | 核心问题 | 行业适应性 |
|:----:|:----:|----------|:----------:|
| 🏛️ **标的质地** | 40% | 这是一个好生意吗？ | ✅ 全行业自动路由 |
| 🌊 **宏观轮动** | 30% | 当下的风往哪吹？ | ✅ 全行业 |
| 📈 **技术趋势** | 30% | 图形告诉你什么？ | ✅ 全行业 |

### 视角 1：标的质地 — 行业自适应评分

**设计理念**：不同的行业有不同的"好生意"标准。Triview 根据行业分类自动路由到最合适的评分分支。

```
行业分类 → 自动路由
──────────────────────────────────────────────────
制造业/科技/硬件  → 供应链卡位分析（六步法）
消费/品牌         → 品牌护城河 + 定价权 + 渠道壁垒
金融/保险         → 网络效应 + 息差管理 + 监管护城河
医疗/生物科技     → 管线价值 + 专利壁垒 + FDA/CFDA 准入
平台/互联网       → 双边网络效应 + 数据壁垒 + 切换成本
能源/大宗商品     → 资源禀赋 + 成本曲线位置
其他（兜底）      → 通用财务质量评分
```

#### 分支 A：供应链卡位（制造业/科技/硬件）

**设计来源**：原有的卡位猎手方法论。适用于所有涉及实物供应链的行业。

**六步量化法**：
```
① 信号去噪 → ② 财务映射 → ③ 有效性验证 → ④ 价值评估 → ⑤ 确认链 → ⑥ 阿尔法评分
```

**评分因子 (原始 0-100)**：
- AI/机器人/EV 链命中 (0-20)
- 不可替代性：毛利率 + R&D 强度 + 供应链分层 (0-25)
- 证据阶梯 (0-15)：strong(公告财报) > medium(研报) > weak(叙事打折)
- 小盘弹性 (0-15)：小市值 + 在链 → 高弹性加分
- 财务健康：收入增长 + 现金流 (0-15)
- 需求拐点：催化剂 + 增长趋势 (0-10)

**供应链八层分层**（越上游系数越高）：
```
材料耗材(1.00) > 制程/封装(0.92) > 设备/测试(0.85) > 芯片/器件(0.78)
> 基础设施(0.70) > 模块/子系统(0.62) > 系统集成(0.50) > 下游需求(0.40)
```

**八类罚分因子**（上限扣 60%）：炒作无订单、微盘流动性、会计/杀猪盘、治理风险、周期陷阱、替代风险、地缘风险、稀释风险

#### 分支 B：品牌护城河（消费/品牌）

**评分因子 (0-100)**：品牌力量(25) + 规模渠道(20) + 增长复购(20) + 财务健壮性(20) + 股息防御(15)

#### 分支 C：网络+监管壁垒（金融/保险）

**评分因子 (0-100)**：网络+规模(30) + 盈利能力(25) + 增长(20) + 资本充足(15) + 股息回报(10)

#### 分支 D：管线+专利壁垒（医疗/生物）

**评分因子 (0-100)**：研发强度(30) + 毛利率定价权(25) + 管线成长(20) + 财务稳定(15) + 规模背书(10)

#### 分支 E：网络效应+数据壁垒（平台/互联网）

**评分因子 (0-100)**：网络效应强度(30) + 盈利模式(25) + 增长动能(20) + 现金流质量(15) + 护城河宽度(10)

#### 分支 F：通用质地评分（兜底）

当无法确定行业分类时的兜底逻辑，综合评估盈利质量、增长、规模、现金流和防御性。

### 视角 2：宏观轮动

**评分因子 (0-100)**：
- 周期匹配 (0-35)：行业与当前宏观象限的适配度
- 行业趋势 (0-30)：收入增长 + 利润率趋势
- 资金流向 (0-20)：代理指标
- 政策方向 (0-15)：产业政策支持度

**宏观六象限**：宽松初期 → 宽松中期 → 过热期 → 紧缩初期 → 紧缩中期 → 衰退期

### 视角 3：技术趋势

**评分因子 (0-100)**：
- 趋势方向 (0-30)：MA 排列
- 动量信号 (0-25)：RSI 位置
- 量价配合 (0-20)：成交额验证
- 波动性 (0-15)：ATR 入场
- 乖离率 (0-10)：均线回归

## 共识引擎

### 加权公式
```
最终分 = 标的质地 × 0.40 + 宏观轮动 × 0.30 + 技术趋势 × 0.30
```

### 信号规则
| 条件 | 信号 |
|------|:----:|
| 两维以上看多 + 总分 ≥ 80 | 🟢 强烈看多 |
| 两维以上看多 + 总分 ≥ 65 | 🟢 看多 |
| 两维中性 + 总分 40-64 | 🟡 中性 |
| 两维以上看空 | 🔴 看空 |
| 任一关键维度 < 30 | 🔴 警告 |

## 机械级自检门禁

每次输出前自动运行 16 项检查：数据完整性(4)、内容质量(4)、一致性(4)、安全(4)。Critical 不通过→报告无效，warning→标记提醒。

## 设计哲学

1. **独立视角，加权共识** — 不让单一方法论主导结论
2. **行业自适应** — 不同行业走不同评分分支，不偏科
3. **可证伪** — 每个得分点都有明确触发条件和证据等级
4. **自检优先** — 宁可不出结论，也不出有漏洞的结论
5. **透明评分** — 每个维度的评分因子、权重、罚分都可见
