# Supply Chain Bottleneck Methodology (供应链卡点方法论)

基于 Serenity (@aleabitoreddit) 的公开研究方法 + UZI-Skill (wbh604) 的严谨化升级提炼。

## 核心逻辑

大行情里真正有价值的机会，常常藏在系统扩张时最难绕开的关键环节。

**不是追逐最显眼的 AI 龙头，而是沿着超大规模云厂商资本开支、ASIC/GPU/TPU、光通信、激光器、衬底、外延片、原材料和设备一路向上游追踪，寻找"小市值、低关注、但下游无法绕开的物理卡点"。**

## 六步分析法（Formal Six-Step Method）

### 第一步：信号去噪（Signal Denoising）
**问题**：这个需求是真实的吗？还是只是叙事？

区分**真实需求** vs **叙事炒作**：
- 真实需求：能落到具体 AI 工作负载（训练/推理/光互连/HBM/液冷/电源）的 BOM，有客户 roadmap、长协、产能认证、设计导入背书。
- 叙事炒作：只有新闻稿、研报标题、"AI 概念"标签，落不到具体料号/客户。

**检查清单**：
- [ ] 超大规模云厂商（Hyperscaler）的 Capex 指引是否在上调？
- [ ] 下游的资本开支是否真实发生？（看财报、招投标、设备订单）
- [ ] 是长期结构性需求，还是短期库存周期？
- [ ] 需求能否映射到具体料号/客户/roadmap？

**输出**：`demand_real | demand_narrative`，narrative 直接降权或淘汰。

### 第二步：映射财务科目（Map to Financials）
**问题**：这种"真实需求"会最先体现在哪个财务科目上？

**翻译路径**：
- 需求 → **营收**（量 × ASP）？
- 供给紧张 → **毛利率**扩张（ASP 提价）？
- 长协锁定 → **经营现金流**改善（预付款/产能预订）？
- 扩产 → **在建工程/CapEx** 增加？

**输出**：明确指向 1-2 个**领先科目**，作为后续验证链的观测口。

### 第三步：寻找瓶颈（Bottleneck Hunting）
**问题**：沿着供应链向上游追溯，哪个环节最难扩产？

**瓶颈特征**：
- ✅ 供应商数量极少（≤5 家全球供应商）
- ✅ 扩产周期长（≥12 个月）
- ✅ 客户认证严格（认证周期 ≥6 个月）
- ✅ 技术/设备壁垒高（需要特殊设备、工艺、配方）
- ✅ 材料纯度/性能要求极高（良率难以快速提升）

**猎被错误分类的小盘（Hunt Misclassified Small-Caps）**：
- 市场仍贴旧标签（"老半导体/小众材料周期股/利基设备商"），未按"AI 卡位股"定价。
- 小盘 + 单一卡位环节 → 需求变化对其营收/EPS 弹性极高（非多元化大盘可比）。
- 输出：候选 ticker + `current_label`（当前市场叙事标签）。

**典型瓶颈环节**：
| 产业链 | 瓶颈环节 | 代表公司 |
|--------|---------|---------|
| AI 光模块 | InP 衬底、激光器芯片 | AXTI, SIVE |
| CPO 光互联 | EAM 激光器、MPO 连接器 | 源杰科技, 炬光科技 |
| HBM 内存 | 存储芯片、TSV 封装 | MU, SK Hynix |
| 电力设备 | 特高压换流变压器、智能调度 | 国电南瑞, 特变电工 |
| 半导体设备 | 薄膜沉积、光刻机核心零部件 | 中微公司, 北方华创 |

### 第四步：验证错误定价（Verify Mispricing）
**问题**：市场是否仍然在用旧叙事给这个标的定价？

确认 relabel（重新定价）是否**持久、可测**：
- **持久**：错误分类不是一两个季度的噪声，而是市场认知滞后于供应链事实。
- **可测**：存在可观测的"市场认知校正"路径（卖方覆盖切换、指数纳入、大客户官宣）。

**输出**：`mispricing_durable: yes/no` + relabel 触发路径清单。

### 第五步：建验证链（Build Validation Chain）
**问题**：怎样确保持续跟踪瓶颈逻辑是否还成立？

建立**可证伪的检查点**，逐期跟踪：
- **财报**：领先科目（第二步）是否如期兑现（营收/毛利/现金流拐点）。
- **Backlog/长协**：在手订单、长协覆盖率、预付款。
- **ASP**：供给紧应推动提价；ASP 走平/下行 = 瓶颈松动预警。
- **产能/CapEx**：自身及竞品扩产进度——竞品扎堆扩产 = 瓶颈将被填平。
- **客户 roadmap**：大客户下一代产品是否仍把它写进 BOM；有无替代材料/工艺量产。

**输出**：检查点清单 + 每项的"绿/黄/红"状态判据。

### 第六步：阿尔法评分（Alpha Scoring）
**问题**：给这个标的的卡位强度打分，决定是否值得投资。

#### Alpha 5-Dimension Scoring

每维 0-5（×20 换算 0-100）：

| 维度 | 含义 | 0-5 打分锚点 |
|---|---|---|
| **确定性 certainty** | 需求/卡位是真的吗？ | 0=纯叙事；3=有客户背书但未放量；5=已设计导入+长协 |
| **清晰度 clarity** | 卡位逻辑是否单一可讲清？ | 0=逻辑绕需多重假设；3=逻辑成立但有杂音；5=一句话讲清卡谁脖子 |
| **纯度 purity** | 该业务占比/是否被稀释？ | 0=卡位业务<10%；3=约一半；5=纯卡位标的(>80%) |
| **弹性 elasticity** | 需求兑现对EPS/估值的弹性？ | 0=大盘低弹性；3=中等；5=小盘单环节、极高弹性 |
| **时间窗 timeframe** | 错误定价还能存在多久？ | 0=已 relabel；3=1-2季度窗口；5=市场尚未察觉、窗口>1年 |

> certainty / clarity / purity 太低 → 一票否决式降级。
> elasticity / timeframe 决定"值不值得重仓 + 现在是不是好时点"。

**综合分 (0-100) = 五维均值 × 20**

#### Evidence Ladder（证据阶梯）

| 级别 | 证据类型 | 打分系数 |
|---|---|---|
| **强 strong** | 公告/财报/交易所文件/客户**定点**/**量产**交付/认证/专利/长协订单 | ×1.0 |
| **中 medium** | 权威财经媒体/卖方研报(有可见假设)/行业协会数据 | ×0.85 |
| **弱 weak** | KOL/论坛/截图/不具名渠道/仅情绪面 | ×0.70 (打折) |

> **铁律**：每个标的至少要有 **1 条强/中证据**，否则卡位分打七折。
> 自媒体内容只能当线索(lead-gen)，最终判断必须靠硬证据。

#### 8 Penalty Factors（八罚分因子）

卡位分数会被以下雷扣分（封顶 60% 折扣）：

| 罚分 | 触发条件 |
|---|---|
| 炒作无订单 | 在链+高热度+零硬证据（典型拉高出货） |
| 微盘流动性 | 市值<30亿（出不来货） |
| 会计/杀猪盘 | trap 检测不安全 |
| 治理 | 实控人质押爆雷 |
| 周期性 | 纯周期行业（不是真成长瓶颈） |
| 替代设计风险 | 技术路线之争/被替代 |
| 地缘 | 出口管制/制裁但**无**国产替代叙事（有国产替代反而是thesis） |
| 稀释 | 定增/增发/解禁 |

#### 供应链 8 层分层（Supply Chain 8-Tier Hierarchy）

越上游、越靠近物理瓶颈，卡位分加成越大：

```
材料耗材(1.00) > 制程/封装(0.92) > 设备/测试(0.85) > 芯片/器件(0.78)
> 基础设施(0.70) > 模块/子系统(0.62) > 系统集成(0.50) > 下游需求(0.40)
```

> 取**最上游**命中的层（瓶颈理论：上游卡点主导定价权）。

#### 卡位评级（Bottleneck Rating）

| 综合分 | 评级 | 操作 |
|--------|------|------|
| **≥80** | 强 | 难替代+供给见底+未被定价 → **重仓候选** |
| **60-79** | 中 | 两条成立 → **试仓 + 跑验证链** |
| **40-59** | 弱 | 仅一条成立 → **观察，不进场** |
| **<40** | 无 | 可替代/供给充足/已relabel → **skip** |

## 验证不可替代性（Verify Irreplaceability）

**问题**：这个公司真的是"卡脖子"节点吗？还是只是蹭热点？

**验证方法**：
1. **客户依赖度**：下游客户是否只有少数几家可选供应商？
2. **认证壁垒**：新进入者需要多长时间才能通过客户认证？
3. **产能弹性**：即使价格上涨，这个公司能多快扩产？
4. **技术护城河**：是否有专利、工艺秘密、设备定制化？

**红旗信号**：
- 🚩 大客户正在认证第二供应商
- 🚩 扩产周期短（<6 个月），竞争对手可以快速跟进
- 🚩 产品是"标准化元器件"，客户可以随时切换供应商
- 🚩 公司自己说"我们是卡脖子"，但没有客户认证证据

## 财务翻译（Financial Translation）

**关键指标**：
| 指标 | 健康信号 | 风险信号 |
|------|---------|---------|
| 收入增速 | >行业平均，且可持续 | 单季度暴增后不可持续 |
| 毛利率 | >30%，且稳定或上升 | <20%，或快速下降 |
| 客户集中度 | 有龙头客户，但不依赖单一客户 | 单一客户占比 >50% |
| 资本开支 | 与收入增长匹配 | 远超收入增长（可能产能过剩） |
| 现金流 | 经营现金流为正 | 收入增长但现金流为负（可能应收账款高） |

**估值判断**：
- 瓶颈公司可以享受"稀缺性溢价"，但不能脱离基本面太远
- 参考：P/S（市销率）是否远超历史和同行？
- 警惕：ATM 增发、可转债、认股权证带来的稀释风险

## 催化剂与失效条件（Catalyst & Invalidation）

**催化剂（正面）**：
- 财报超预期（收入/利润/指引）
- 大客户新增订单或延长合作
- 政府补贴/产业政策支持
- 交易所提升（如新三板 → 北交所 → 科创板/创业板）
- 被纳入指数（如 MSCI、沪深 300、标普 500）
- 空头仓位过高后的空头回补

**失效条件（负面）**：
- 出现可量产的替代材料/替代工艺，切换成本崩塌
- 行业大规模扩产公告扎堆 → 瓶颈即将被填平 → 供需反转
- 股价已完成 relabel（市场已按"AI 卡位股"重新定价），alpha 被吃掉
- 需求侧叙事被证伪（AI capex 砍单/客户去库存）
- 宏观冲击（加息、衰退、地缘冲突）

## 案例研究

### 案例 1：AXTI（AXT Inc）—— InP 衬底瓶颈（走一遍六步法）

**① 信号去噪**：AI 数据中心 → 800G/1.6T 光模块放量 → 激光器需要 **InP 衬底**。
落到具体 BOM 与客户 roadmap → `demand_real`。（非纯概念）

**② 映射财务科目**：需求 → InP 衬底**营收量增** + 供给紧带动 **ASP / 毛利率**上行；
扩产体现在 **capex / 在建工程**。领先科目锁定：营收（InP 分部）+ 毛利率。

**③ 猎 misclassified 小盘**：市场当时把 AXTI 当"小众化合物半导体周期股"定价（
current_label = "老周期半导体小盘"），未按"AI 光互连卡位股"定价；小盘 + 单环节 → 高弹性。

**④ 验证错误定价**：InP 高品质衬底全球量产玩家极少、产能紧 → 错误分类持久；
relabel 路径 = AI 光模块需求被市场认知 + 卖方覆盖切换。`mispricing_durable: yes`。

**⑤ 建验证链**：
- 财报：InP 分部营收/毛利率拐点（绿）
- backlog：光模块客户长协/预订（黄→绿）
- ASP：衬底提价趋势（绿）
- 产能：自身扩产 ramp 节点 + 监控竞品是否扎堆扩产（黄，重点盯）
- roadmap：客户下一代是否仍用 InP、有无替代衬底材料量产（绿）

**⑥ 阿尔法评分（示意）**：

| 维度 | 分(0-5) | 理由 |
|---|---|---|
| 确定性 certainty | 4 | 需求真实、客户背书，放量中 |
| 清晰度 clarity | 5 | "光模块绕不过 InP 衬底"一句话讲清 |
| 纯度 purity | 4 | InP 是核心弹性来源，但仍有 GaAs 等其它业务 |
| 弹性 elasticity | 5 | 小盘单环节，EPS 弹性极高 |
| 时间窗 timeframe | 5 | 进场时市场尚未 relabel，窗口 >1 年 |

综合 = (4+5+4+5+5)/5 ×20 = **92 → 卡位评级「强」→ 重仓候选**

**结果回溯**：随 AI 光模块需求被市场认知，股价由约 **$12 → $70+**，
完成 relabel——按 bearish 规则（relabel 完成/警惕竞品扩产填平瓶颈）应进入减仓观察。

### 案例 2：AAOI（Applied Optoelectronics）—— CPO 激光器供应商
**背景**：CPO（Co-Packaged Optics）是下一代 AI 数据中心互联方案，需要大量激光器。

**瓶颈逻辑**：
- CPO 需要 EAM（电吸收调制器）激光器，AAOI 是主要供应商
- 客户：NVIDIA、微软、Meta 等超大规模云厂商
- 产能扩张跟不上需求增速

**风险**：
- 大客户正在认证第二供应商（如 LITE、COHR）
- CPO 技术路线尚未统一，存在技术迭代风险

### 案例 3：源杰科技（688498.SH）—— A 股 InP 激光器芯片
**背景**：A 股 AI 光模块产业链受益标的。

**瓶颈逻辑**：
- 25G/50G/100G PON 激光器芯片国产替代
- 客户：中际旭创、华工科技等光模块龙头
- 产能扩张受限（需要 MOCVD 设备，交货周期长）

**风险**：
- 估值已充分定价（P/S >100x）
- 下游光模块价格战可能传导到上游芯片
- 大客户集中度高风险

## 常见陷阱

### 陷阱 1：把"热点"当"瓶颈"
- ❌ 错误："AI 很热，所以所有 AI 概念股都会涨"
- ✅ 正确："AI 导致 HBM 需求暴增，而 HBM 产能扩张需要 18 个月，所以 HBM 供应商是瓶颈"

### 陷阱 2：忽略客户认证周期
- ❌ 错误："这家公司说要做 InP 衬底，所以它是瓶颈"
- ✅ 正确："这家公司有没有通过下游光模块公司的认证？认证周期需要多久？"

### 陷阱 3：把"短期缺货"当"长期瓶颈"
- ❌ 错误："现在 InP 衬底缺货，所以 AXTI 可以一直涨"
- ✅ 正确："InP 衬底产能扩张周期多长？18 个月后供需是否会逆转？"

### 陷阱 4：忽略估值已充分定价
- ❌ 错误："AXTI 是瓶颈，所以现在任何价格都可以买"
- ✅ 正确："AXTI 的 P/S 已经从 2x 膨胀到 15x，alpha 空间已被压缩"

### 陷阱 5：忽略政策/地缘政治风险
- ❌ 错误："这家公司技术领先，所以没问题"
- ✅ 正确："这家公司的设备/原材料是否需要出口许可证？地缘政治紧张会不会导致供应链中断？"

## Serenity 卡位链覆盖范围（v3.7+ 扩展）

Serenity 的"AI 链"不限于半导体/光通信——只要符合卡脖子逻辑就纳入：

| 子链 | 代表卡位环节 |
|---|---|
| **数据中心算力光互连** | 光模块/CPO/硅光/InP磷化铟衬底/HBM/CoWoS/高速铜连接/液冷 |
| **AI 终端侧光学** | 光波导/衍射光波导/Micro-LED/硅基OLED/晶圆级光学(AR/VR近眼显示) |
| **具身智能/人形机器人** | 谐波减速器/RV减速器/行星滚柱丝杠/灵巧手/空心杯电机/六维力传感器/电子皮肤 |

> **为什么纳入具身智能**：「具身智能 = AI 的物理化」。人形机器人的上游核心零部件
> （如**谐波减速器**——绿的谐波 688017）同样满足"难替代 + 供给紧 + 没被定价"。
> ⚠️ 注意：大盘整机厂（切换成本低、规模壁垒弱）即使命中"机器人"也只给中性，
> **卡位 gating（不可替代 + 中小盘 + 需求拐点）仍是决定性的**。

## 检查清单（Checklist）

分析一只股票时，按此清单逐项检查：

**需求确认（Signal Denoising）**：
- [ ] 下游客户的 Capex 指引是否在上调？
- [ ] 是否有真实的订单/财报验证需求？
- [ ] 是结构性需求还是短期库存周期？
- [ ] 需求能落到具体 BOM/客户 roadmap 吗？

**瓶颈识别（Bottleneck Hunting）**：
- [ ] 这个环节的全球供应商是否 ≤5 家？
- [ ] 扩产周期是否 ≥12 个月？
- [ ] 客户认证周期是否 ≥6 个月？
- [ ] 是否有技术/设备/工艺壁垒？
- [ ] 市场是否还在用旧叙事给它定价？（misclassified？）

**不可替代性（Irreplaceability）**：
- [ ] 下游客户是否只有少数几家可选供应商？
- [ ] 新进入者需要多长时间才能通过认证？
- [ ] 即使价格上涨，公司能多快扩产？
- [ ] 是否有专利/工艺秘密/设备定制化护城河？

**财务质量（Financial Translation）**：
- [ ] 收入增速是否可持续？
- [ ] 毛利率是否 >30% 且稳定？
- [ ] 客户集中度是否过高（单一客户 >50%）？
- [ ] 经营现金流是否为正？

**证据力度（Evidence Ladder）**：
- [ ] 是否有 ≥1 条**强**或**中**级证据？
- [ ] 自媒体内容是否已标注为 weak（×0.70）？
- [ ] 是否有 8 罚分因子被触发？

**估值判断（Alpha Scoring）**：
- [ ] 5 维 alpha 评分是否 ≥60（进入"中"评级）？
- [ ] 供应链层级加成（8 层 hierarchy）是否已应用？
- [ ] P/S 是否远超历史和同行？
- [ ] 是否有 ATM 增发/可转债/认股权证稀释风险？

**催化剂 & 失效条件**：
- [ ] 未来 3-6 个月有哪些可能的催化剂？
- [ ] 什么情况下这个瓶颈逻辑会失效？
- [ ] 替代品量产/竞品扩产/relabel 完成等 bearish 信号是否已出现？

## 总结

供应链卡点分析不是"找热点"，而是：

1. **沿供应链向上游追溯**
2. **找到最难扩产、最难替代的环节**
3. **用六步法验证（信号去噪→财务映射→猎小盘→验证错误定价→建验证链→阿尔法评分）**
4. **用证据阶梯判别证据可信度**
5. **用 8 罚分因子检查隐患**
6. **用 8 层分级加权卡位位阶**
7. **跟踪催化剂和失效条件**

**最重要的原则**：价格从来不是第一步。先搞清楚"供需缺口还在不在"，再谈价格。
